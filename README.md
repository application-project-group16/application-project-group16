###  Description of the project
A mobile application project for school by group 16.
The app's idea is to offer users a service to search for sports friends with similar interests

Main ideas and concepts:
  - The user can log in and log out
  - The user can swipe through different profiles, filter them, and review profiles before swiping
  - User can search for the nearest sports places, like gyms, using the Overpass API
  - The user can review statistics about their matches and the app's users
  - Friend users can chat with each other
  - The user can edit their own profile 

Monetization ideas:
  - If the user isn't a premium member, they will receive ads, for example, between swipes. Features are also limited
  - Premium members receive an ad-free version and full access to the features of the apps
